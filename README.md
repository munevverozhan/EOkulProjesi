# EOkulProjesi
Visual Studio ortamında C# dili ve .NetFrameworkü kullanarak geliştirdiğim bu projede; öğrenci ve öğretmen girişi olan, öğretmen'in giriş yapınca gerçekleştirebildiği çeşitli işlemler bulunan ve öğrenci giriş yapınca notlarını listeleyebilen bir proje tasarladım.
